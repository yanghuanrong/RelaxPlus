# Icon 图标
> 语义化的矢量图形

## 演示
> 直接通过设置类名为 `x-icon-iconName` 来使用即可。例如：
:::demo
```html
<template>
  <x-icon type="x-icon-star-on"></x-icon>
  <x-icon type="x-icon-link"></x-icon>
</template>
```
:::
